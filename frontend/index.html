Pre-Flight Anomaly Detection

by Anthony Ricevuto

This is a demo frontend for the Pre-Flight Anomaly Detection service. It sends randomized sample sensor data to the anomaly-detection function and displays the response (anomalies, normal ranges, and detected issues).

This project is engineered to abide by NASA's Software Engineering Procedural Requirements, Standards, and Related Resources (NPR 7150.2, NASA-STD-8739.8, the NASA Secure Coding guidance, and the NASA/JPL "Power of Ten" rules).
